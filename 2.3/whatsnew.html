New Features

CDI injection

Various JSF artifacts are now @Inject-able. See the list below for which ones are. Note at the moment this is enabled by creating a faces-config.xml in your web application with version 2.3.

  • applicationMap
  • externalContext
  • facesContext
  • session (delegating that responsibility back to default CDI runtime)
  • sessionMap
  • view
  • viewMap
